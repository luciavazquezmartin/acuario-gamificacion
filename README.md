# acuario-gamificacion
Diseño y prototipado de sistema interactivo gamificado para el Acuario de Zaragoza. Incluye interfaces adaptativas por edad (UX/UI) y mecánicas de yincana educativa.

# Instrucciones de Prueba

Para desbloquear las funciones de Administrador y configuración avanzada en el prototipo, sigue estos pasos:

# 1. Acceso al Panel de Control:

    Ve a AJUSTES > CONFIGURACIÓN AVANZADA.

    Truco: Debes seleccionar de nuevo el título para que salte el login.

# 2. Credenciales de Acceso: Usa estos datos ficticios para entrar:

    Usuario: empleado1@acuario.es Contraseña: aquakidszgz

# 3. Notas para la Simulación:

    Atajos: En la pantalla principal verás un botón "FIN". Úsalo para saltar directamente al final del juego y ver las pantallas de conclusión sin jugar toda la partida.

    Actividad Grupal: Si eliges el rol "PADRE", verás la interfaz de gestión. Si eliges "NIÑO", verás la interfaz de juego (controlada por el padre).
